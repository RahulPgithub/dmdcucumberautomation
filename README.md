# DMD API Automation Framework

A comprehensive, scalable Cucumber automation framework for testing DMD Microservices (DMD Features Service and DMD DeviceInfo Service) using REST Assured and Maven.

## Features

- **Multi-Microservice Support**: Tests for DMD Features Service and DMD DeviceInfo Service
- **XML Payload Support**: Load and execute XML payloads for API requests
- **REST Assured Integration**: Easy HTTP method support (GET, POST, PUT, DELETE)
- **Multiple Environments**: Support for dev, qa, staging, and production environments
- **Comprehensive Reporting**: Cucumber HTML reports + custom test results output
- **Detailed Logging**: Log4j2 integration for request/response logging
- **Extent Reports**: Advanced reporting with pass/fail status tracking
- **GitLab CI/CD**: Automated pipeline execution on code commits
- **Scalable Architecture**: Easy to add new APIs and services
